# slipstream-server example

This directory provides a minimal slipstream-style server in Go.

## Requirements
- Go 1.20+

## Install

```bash
cd /workspaces/Tester/slipstream-server
go mod init github.com/Bunya7388/Tester/slipstream-server
```

## Run server

```bash
go run ./server.go
```

## Run client (in another shell)

```bash
go run ./client.go
```
